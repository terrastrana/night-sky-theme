Celestial

A Tumblr theme inspired by the night sky.

Download this project as a .zip file Download this project as a tar.gz file

Celestial is a basic Tumblr theme using the night sky as an inspiration. Here is a working example of the theme: night-sky-theme.tumblr.com.

Notes for Customizing the Theme: The image used in the sidebar was originally taken from the following link: http://www.wallconvert.com/wallpapers/photography/starry-sky-over-the-observatory-18232.html. The image has been included in the night-sky-theme repository for your convenience. Please note that the sidebar will display a blank black box until a Sidebar Image has been uploaded.

The Background Image setting is optional, and will display as a tiled image if activated.
